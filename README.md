# Movie Plot Gender Bias Analysis
## Network Science and Machine Learning Approach

This project aims to quantify and visualize gender bias within movie plot summaries using a combination of Natural Language Processing (NLP), Network Science, and Machine Learning. By representing narratives as knowledge networks, we analyze how character roles and conceptual associations differ across genders.
